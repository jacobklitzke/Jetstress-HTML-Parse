This code will parse the HTML Jetstress files generated by Jetstress, and perform some basic calculations on those values.
The information that is output by this program is used for benchmarking puposes.

To run this file, copy the file to your desktop, open a command prompt, and type the following commands: "

cd Desktop
java -jar JetstressParse.jar

You will need to enter the path to the stress, backup, and soft recovery files. 
You will also need to enter the number of databases you used when you ran the test.
